Новая Библиотека Московского Политеха

Цель проекта: создание комфортной библиотечной среды, с применением новых цифровых технологий, встроенной в современный образовательный процесс для эффективной самостоятельной и групповой работы, саморазвития, общения и обмена опытом между участниками образовательного процесса.

Подгруппа «Мероприятия»

Задача подгруппы

Разработка мероприятий с целью привлечения студентов к посещению библиотеки Московского Политеха. В создание мероприятий входят:

  • написание сценариев;
  • создание публикационных постов для VK и Instagram;
  • проведение меропприятий на уровне проекта.

Среди десятков предложенных мероприятий для проработки на начальном этапе работы подгруппы были выбраны следующие:

  • «Литературная викторина»;
  • «Литературный клуб»;
  • «Вечер кино»;
  • «Открытый микрофон».


Литературный клуб


Мероприятие создано для поиска единомышленников с целью обсуждения книг в рамках библиотеки.

Литературная викторина


Мероприятие отлично подходит для проведения викторин символично праздникам, связанным с библиотечной тематикой.


Вечер кино


Специально для киноманов. Обсудить экранизации, а также провести параллели с книгой можно только на данном мероприятии.

Открытый микрофон


Мероприятие проводится для совершенно разных студентов, которым есть, что сказать. Прочесть стих, спеть песню, выступть со stand-up шоу можно посетив «Открытый микрофон».

Ход работы

В течение семестра подгруппа «Мероприятия» активно занималась составлением сценариев, написанием постов и проведением каждого мероприятия. Деятельность участников подгруппы началась с определения списка событий, которые студентам хотелось отразить в ходе проекта. С идеями мероприятий можно ознакомиться, перейдя по ссылке:

Идеи мероприятий...

Подгруппа разбилась на несколько коллективов для каждого мероприятия, которые были выбраны для разработки. В каждой из полученных подгрупп было выбрано ответственное лицо, занимающееся отчетностью перед руководителем подгруппы «Мероприятия», организацией работы студентов при разработке мероприятия – написанием сценариев и постов для социальных сетей. Для каждого мероприятия также был составлен список необходимых предметов для проведения и организационных моментов.


На уровне проекта студентами были протестированы все мероприятия, которые выбирались для проведения в начале семестра. С материалами, разработанными в течение семестра каждой из подгрупп, можно ознакомиться по ссылке:

Наработки по мероприятиям...

Обучающиеся, входящие в состав подгруппы «Мероприятия», активно взаимодействовали с другими участниками проекта, благодаря чему на первых этапах разработки мероприятий была получена обратная связь, направленная на улучшение организационных моментов касаемо проведения. Помимо проведения мероприятий перед участниками проекта также были поставлены следующие задачи, с которыми студенты успешно справились:

  • предложить варианты организации обратной связи через Вконтакте и Instagram, которые доступны по ссылке;
  • разобрать вариант обратной связи через создание QR-кодов;
  • разобрать вариант обратной связи путем заполнения Google-формы с интеграцией в Google-таблицу;
  • прописать рейтинговую систему для участников проекта.

Участниками подгруппы также был сформирован первичный календарь мероприятий на осенний семестр 2021 года, с которым можно ознакомиться по ссылке:
